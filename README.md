

# 🎓 Smart Student Placement Registration System

## 📌 Project Description

The **Smart Student Placement Registration System** is a web-based application designed to simplify and automate the campus placement process. It allows students to register their academic details, skills, and upload resumes, while administrators can manage, filter, and shortlist candidates efficiently.

---

## 🚀 Key Features

* 🔐 Secure Login & Registration (Student/Admin)
* 📄 Student Profile Management (Skills, CGPA, Resume Upload)
* 📊 Admin Dashboard to View and Manage Students
* 🔍 Filter Candidates Based on Skills and Performance
* ✅ Shortlisting of Eligible Students
* 📧 Automated Interview Call Letter Generation (PDF + Email)

---

* Nodemailer

---

## 📊 System Workflow

1. Students register and submit their details
2. Data is stored in the database
3. Admin accesses dashboard to view all students
4. Candidates are filtered based on skills/CGPA
5. Eligible students are shortlisted
6. Interview call letters are generated and sent via email
